# crosslex-next

The main Crosslex web app, built with Next.js 14 (pages router).

## Development

Run from `website/`:

```bash
yarn dev:crosslex-next
```

The app runs on [http://localhost:3000](http://localhost:3000).

## Build

```bash
yarn workspace @whitelotus/crosslex-next build
```

## Lint

```bash
yarn workspace @whitelotus/crosslex-next lint
```
